# 枚举

- 可以被穷举的结构，有限个数

- 使用 **<font color="orange"> enum </font>** 关键字来声明，与类同级。

```C#
enum <enum_name>
{ 
    elements list 
};
```

- 枚举本身可以有修饰符，但枚举的成员始终是公共的不能有访问修饰符。

- 枚举本身的修饰符仅能使用 **<font color="orange"> public </font>** 和 **<font color="orange"> internal </font>**.

- 枚举是一个指定的常数，其基础类型可以是除 Char 外的任何整型。
  
- 如果没有显式声明基础类型，则使用 Int32。

- 编程语言通常提供语法来声明由一组已命名的常数和它们的值组成的枚举

- 默认基数从O开始，也可指定数值

```C#
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace Lesson01
{
    enum Days       // enum type with default value
    {
        Monday,
        Tuesday,
        Wensday,
        Thursday,
        Friday,
        Saturday,
        Sunday    // enum elements
    };

    enum Days1       // enum type with specified value
    {
        Monday=1,
        Tuesday,
        Wensday,
        Thursday,
        Friday,
        Saturday,
        Sunday    // enum elements
    };
    enum Colors 
    { 
        Red = 1, 
        Green = 2, 
        Blue = 4, 
        Yellow = 8 
    };

    public enum NoticeType
    {
        Notice = 'A',
        LabRule = 'H',
        HotInformation = 'N',
        Column = 'C',
        All = '1',
        Null = '0'
    }

    class FirstClass
    {
        static void Main(string[] args)
        {
            Console.WriteLine("\n\nModay without value : ");
            Console.WriteLine("Monday default is " + (int)Days.Monday);
            Console.WriteLine("Tuesday default is " + (int)Days.Tuesday);
            Console.WriteLine("Wensday default is " + (int)Days.Wensday);
            Console.WriteLine("Thursday default is " + (int)Days.Thursday);
            Console.WriteLine("Firday default is " + (int)Days.Friday);
            Console.WriteLine("Saturday default is " + (int)Days.Saturday);
            Console.WriteLine("Sunday default is " + (int)Days.Sunday);

            Console.WriteLine("\n\nModay is specified 1 : ");
            Console.WriteLine("Monday default is " + (int)Days1.Monday);
            Console.WriteLine("Tuesday default is " + (int)Days1.Tuesday);
            Console.WriteLine("Wensday default is " + (int)Days1.Wensday);
            Console.WriteLine("Thursday default is " + (int)Days1.Thursday);
            Console.WriteLine("Firday default is " + (int)Days1.Friday);
            Console.WriteLine("Saturday default is " + (int)Days1.Saturday);
            Console.WriteLine("Sunday default is " + (int)Days1.Sunday);


            // 使用
            Colors myColors = Colors.Red;
            string strColor = myColors.ToString();
            int IntColor = (int)myColors;

            // 位或
            myColors = Colors.Red | Colors.Blue | Colors.Yellow;

            // 位与
            myColors = Colors.Red & Colors.Blue & Colors.Yellow;

            // 遍历 
            foreach (string s in Enum.GetNames(typeof(Days)))
                Console.Write("\n\n" + s + "--" + Enum.Parse(typeof(Days), s).ToString());

            //新建枚举类型
            NoticeType noticeType1 = NoticeType.Column;

            //把枚举类型转换为string d="Column"
            string d = noticeType1.ToString();

            //取得枚举类型的基数 dd='C'
            char dd = (char)noticeType1;

            //通过基数取得对应的枚举类型 noticeType2 = NoticeType.Notice
            //(NoticeType)'A';  两种方式都可以
            NoticeType noticeType2 = (NoticeType)Char.Parse("A");

            //通过名称取得枚举类型 noticeType3 = NoticeType.Notice
            NoticeType noticeType3 = (NoticeType)Enum.Parse(typeof(NoticeType), "Notice");

        }
    }
}
```
