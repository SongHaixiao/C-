# 选择排序 （Select Sort）

选择排序改进了冒泡排序，冒泡是比较完就交换，而选择排序则是选出最小的才交换.

### 描述

1. 扫描整个序列

2. 从中挑出最小的数据项

3. 将最小的数据项放置到合适的位置

```
6 5 4 7

1. 假设第一个最小，验证是否最小

2. 记忆最新的最小为 5 的 index = 1

3. 重复以上 2 步到数组末尾

4. 最小的位置被找到 值为 4 的 index = 2

5. 0 索引 和 2 索引交换位置

6. 如此循环选择 n - 1 次，程序结束
```

```C#
using System;
using System.Collections;
using System.Collections.Generic;
using System.ComponentModel.DataAnnotations;

namespace Test1
{
    class Test
    {
        static void Main(string[] args)
        {
            int[] arr = new int[5] { 3, 8, 4, 7, 2 };

            PrintArray(arr, "Origin : ");

            // Select Sort
            for(int i = 0; i < arr.Length - 1; i++)
            {
                // 假设当前的位置正好是最小的
                int minIndex = i;
                for(int j = i + 1; j < arr.Length; j++)
                {
                    if (arr[minIndex] > arr[j])
                    {
                        // 发现了更小的位置
                        minIndex = j;
                    }
                }

                // 确定了最小的位置，交换这两个位置的值
                Swap(ref arr[i], ref arr[minIndex]);
            }

            PrintArray(arr, "Sorted : ");
        }

        public static void Swap(ref int n1, ref int n2)
        {
            // swap
            int temp = n1;
            n1 = n2;
            n2 = temp;
        }

        public static void PrintArray(Array arr, String s)
        {
            Console.WriteLine("\n\n" + s);

            foreach (var item in arr)
            {
                Console.Write(item + " ");
            }
        }
    }    
}
```
```
output:



Origin :
3 8 4 7 2

Sorted :
2 3 4 7 8
```
