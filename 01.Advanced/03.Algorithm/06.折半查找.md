# 折半查找

在有序表中，把待查找的数据值与查找范围的中间元素值进行比较，会有三中情况出现：

1） 待查找数据值与中间元素值正好相等，则返回中间元素值的索引；

2) 待查找数据值比中间元素值小，则以整个查找范围的前半部分作为新的查找范围，执行 1）, 直到找到相等的值；

3） 待查找数据值比中间元素值大，则以整个查找范围的后半部分作为新的查找范围，执行 1），直到找到相等的值；

4） 如果最后找不到相等的值，则返回错误提示信息，一般返回索引 -1 表示未找到.

```C#
using System;
using System.Collections;
using System.Collections.Generic;
using System.ComponentModel.DataAnnotations;

namespace Test1
{
    class Test
    {
        static void Main(string[] args)
        {
            string [] arr = new string[5] { "abc", "def", "glm", "opl", "mnq"};

            PrintArray(arr, "Original Sequence:");

            string target = "aaa";
            int index = BinarySearch(arr, target);

            if (index != -1)
            {
                Console.WriteLine("\nFind the target element {0} in {1} location.", target,index);
            }
            else
            {
                Console.WriteLine("\nCan't find the target element {0}.",target);
            }

            target = "abc";
            index = BinarySearch(arr, target);

            if (index != -1)
            {
                Console.WriteLine("\nFind the target element {0} in {1} location.", target, index);
            }
            else
            {
                Console.WriteLine("\nCan't find the target element {0}.", target);
            }
        }

        public static int BinarySearch(string[] arr, string str)
        {
            // swap
            int left = 0;
            int right = arr.Length - 1;

            while(left <= right)
            {
                int middle = (left + right) / 2;

                if (arr[middle] == str)
                {
                    return middle;
                }
                else if (str.CompareTo(arr[middle]) > 0){
                    left = middle + 1;
                }
                else
                {
                    right = middle - 1;
                }
            }

            return -1;
        }

        public static void PrintArray(Array arr, String s)
        {
            Console.WriteLine("\n\n" + s);

            foreach (var item in arr)
            {
                Console.Write(item + " ");
            }
        }
    }   
}
```
