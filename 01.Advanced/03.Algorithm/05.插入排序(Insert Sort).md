# 插入排序（Insert Sort)

插入排序是简单排序里最好的一种，但是稍微麻烦一些

### 描述

1. 假设部分有序（一般设第一个数据项为第一部分）.
   
2. 其他输入一次插入之前的有序序列.

3. 若序列基本有序，此排序算法最优.

4. 要注意为待插入元素找到合适位置

```
Original Sequence  ： 3 5 7 2 1

New Sequence : select 1th element as first part : 3

continue to select element from Original Sequence : 5

5 > 3 insert New Sequence after 3

New Sequence : 3 5

continue to select element from Original Sequence : 7

7 > 5 insert New Sequence after 5

New Sequence : 3 5 7

continue to select element from Original Sequence : 2

2 < 7 < 5 < 3 insert New Sequence before 3

New Sequence : 2 3 5 7

continue to select element from Original Sequence : 1

1 < 7 < 5 < 3 < 2 insert New Sequence before 2

New Sequence : 1 2 3 5 7

Sorted Sequence : 1 2 3 5 7
```
```C#
using System;
using System.Collections;
using System.Collections.Generic;
using System.ComponentModel.DataAnnotations;

namespace Test1
{
    class Test
    {
        static void Main(string[] args)
        {
            int[] arr = new int[5] { 8, 2, 4, 1, 3 };

            PrintArray(arr, "Origin : ");

            // Insert Sort
            // 假设最开始索引为 0 的位置为一部分且有序
            for(int i = 1; i < arr.Length; i++)
            {
                int temp = arr[i];
                int insertIndex = i;

                while(insertIndex > 0 && arr[insertIndex-1] > temp)
                {
                    // move
                    arr[insertIndex] = arr[insertIndex - 1];
                    insertIndex--;
                }

                arr[insertIndex] = temp;
            }

            PrintArray(arr, "Sorted : ");
        }

        public static void Swap(ref int n1, ref int n2)
        {
            // swap
            int temp = n1;
            n1 = n2;
            n2 = temp;
        }

        public static void PrintArray(Array arr, String s)
        {
            Console.WriteLine("\n\n" + s);

            foreach (var item in arr)
            {
                Console.Write(item + " ");
            }
        }
    }    
}
```

output:



Origin :
8 2 4 1 3

Sorted :
